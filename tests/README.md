# 試験項目表ディレクトリ

このディレクトリは、E2E試験の結果を記録した試験項目表を保存する場所です。

## 目的

- spec.md に記載された試験項目に基づいて実施した試験結果を保存
- 試験履歴の管理・追跡

## ファイル命名規則

- **形式**: `YYYYMMDD###試験項目表.md`
  - `YYYYMMDD`: 試験実施日（例: 20251221）
  - `###`: 3桁の通し番号（例: 001, 002）
- **例**: `20251221001試験項目表.md`, `20251221002試験項目表.md`
- 同一日に複数回試験を実施する場合は通し番号をインクリメントする

## 関連ドキュメント

試験項目の詳細については、[spec.md の 13.3 試験項目表](../versions/v0.3.0/spec.md#133-試験項目表) を参照してください。
